Replication code for "Calibration in Context: A Case Study with Score Decompositions"
================

This supplementary material provides replication code for the workshop paper “Calibration in Context: A Case Study with Score Decompositions” by Johannes Resin presented at the workshop “Towards Trustworthy Predictions: Theory and Applications of Calibration for Modern AI” at AISTATS 2026, Tangier, Morocco.

The data can be obtained from the supplementary material to Gupta and Ramdas (2022, ICLR) at https://openreview.net/forum?id=WqoBaaPHS- and need to be pasted into the data folder before running the code.
